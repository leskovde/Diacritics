# Použití

'`diac' -i`			Instalace - vyžaduje stažený soubor '_diac_model.hzip' v aktuálním adresáři

`'diac' -d`			Demo - vyžaduje stažené soubory demo0\*.txt a demo0\*_ref.txt

`'diac --help'`		Help - zobrazení kompletní nápovědy
